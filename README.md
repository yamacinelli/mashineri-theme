# Installation
1. Open the Extensions sidebar in VS Code
2. Search for Mashineri Theme
3. Click **Install**
4. Open the **Command Palette**:
    - Windows / Linux: Press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>
    - macOS: Press <kbd>Command</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>
5. Select **Preferences: Color Theme** and choose an Mashineri variant.
